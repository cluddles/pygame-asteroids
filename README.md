Very, very, *very* barebones Asteroids clone written in Python/Pygame

* WASD (or arrow keys) to move
* Space to shoot
* Escape to exit application

### How to use

Setup virtual environment:
```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Run:
```
python3 main.py
```
or just `./main.py`
